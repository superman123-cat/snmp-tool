快速开始

  1. 点击左侧「MIB 树」标签的「导入」按钮上传 .mib 文件
  2. 点击「设备」标签添加 SNMP 设备（支持 v1/v2c/v3）
  3. 在 MIB 树中右键节点，选择「查询单个节点」或「查询整表」
  4. 使用顶部搜索框进行全局搜索
